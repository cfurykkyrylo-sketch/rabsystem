# 🚀 rabsystem - Personal Shift & Finance Tracker

**rabsystem** — це настільний WPF-застосунок для обліку робочих змін, розрахунку заробітної плати та відстеження фінансових цілей. Розроблено в рамках навчальної практики.

## 📌 Основний функціонал
- **Журнал змін:** Додавання, перегляд та видалення відпрацьованих змін.
- **Автоматичний розрахунок:** Підрахунок заробітку на основі введених годин та погодинної ставки (77 грн/год).
- **Аналітика:** Відображення загального заробітку, рекордної зміни та прогресу досягнення фінансової цілі.
- **Персоналізація:** Миттєва зміна мови (UA/EN) та тем (Світла/Темна).
- **Збереження даних:** Всі записи надійно зберігаються у локальний файл `shifts.json`.

## 🛠 Технологічний стек
- C# / .NET 8.0 (Desktop Development)
- WPF (Windows Presentation Foundation)
- XAML (з використанням ResourceDictionaries)
- JSON Serialization (для збереження даних)

## 🚀 Інструкція із запуску
1. Завантажте проєкт або склонуйте репозиторій.
2. Відкрийте файл рішення `ShiftTracker.sln` у Visual Studio 2022.
3. Переконайтеся, що встановлено робоче навантаження ".NET desktop development".
4. Оберіть режим конфігурації **Release** або **Debug**.
5. Натисніть `F5` для запуску або `Ctrl+Shift+B` для компіляції `.exe` файлу (знаходиться у папці `bin/Release/net8.0-windows`).
